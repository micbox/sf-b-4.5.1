# PHP style array functions for JS

## First revision:
* array_sum
* array_reverse
* array_merge

## Sample usage

    myArray = [1, 2, 3, 4, 5];

    sum = array_sum(myArray);

    console.log(sum);

    console.log(array_reverse(myArray));

    console.log(array_merge(myArray, [7,8,9], [-4, -7, -99]));
